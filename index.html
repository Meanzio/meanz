항공교육 실적관리 시스템

분산된 교육 이수 데이터와 실적 보고 흐름을 하나의 관리 체계로 정리한 프로젝트입니다.

파일 다운로드

문제 정의

  • 교육 이수 현황과 실적 데이터가 여러 파일에 분산되어 있었습니다.
  • 보고 시점마다 수동 취합이 반복되어 누락과 중복 위험이 있었습니다.
  • 관리자가 대상자, 과정, 이수 상태를 빠르게 파악하기 어려웠습니다.

기획 포인트

  • 교육 과정, 대상자, 이수 상태, 실적 기준을 데이터 단위로 정리했습니다.
  • 검색, 필터, 상세 조회, 다운로드 중심의 관리자 화면을 설계했습니다.
  • 운영자가 검수와 보고에 바로 활용할 수 있는 흐름을 제안했습니다.

교육 플랫폼 서비스

교육 콘텐츠 탐색부터 수강, 이력 관리까지 이어지는 학습 플랫폼 기획 프로젝트입니다.

파일 다운로드

문제 정의

  • 학습자는 필요한 교육을 찾기 어렵고 운영자는 콘텐츠 관리가 어려웠습니다.
  • 신청, 수강, 완료, 이력 확인 과정이 분절되어 있었습니다.
  • 사용자 기능과 관리자 기능의 권한 구분이 필요했습니다.

기획 포인트

  • 홈, 과정 목록, 과정 상세, 수강 현황, 관리자 메뉴로 IA를 정리했습니다.
  • 학습자 탐색 흐름과 운영자 관리 흐름을 분리했습니다.
  • 검색, 카테고리, 수료 기준, 상태값 등 운영 정책을 정의했습니다.

주문수집 자동화 시스템

반복적인 주문 확인과 수기 입력 업무를 줄이고 처리 상태를 추적할 수 있게 만든 프로젝트입니다.

파일 다운로드

문제 정의

  • 여러 채널에서 들어오는 주문을 사람이 직접 확인하고 정리해야 했습니다.
  • 주문 누락, 중복 입력, 처리 상태 불명확 등 운영 리스크가 있었습니다.
  • 관리자가 주문 처리 이력을 한눈에 확인하기 어려웠습니다.

기획 포인트

  • 수집, 검수, 처리, 완료, 예외 상태 기준으로 업무 흐름을 재설계했습니다.
  • 관리자 리스트, 상세 화면, 상태 변경, 다운로드 기능을 구성했습니다.
  • 자동화하면서도 운영자가 예외 상황을 통제할 수 있는 구조를 만들었습니다.
